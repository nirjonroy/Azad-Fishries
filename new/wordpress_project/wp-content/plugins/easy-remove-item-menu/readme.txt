== Description ==

You can remove items from the menu very easily

You don't need to click on an item, then click "Remove" link. Just click "x" button (to delete an item) or click "xx" (to delete an item and all its sub items)

This plugin is an improvement of the "Quick remove menu item" plugin from hpthemes

This plugin has been optimized to work better

== Screenshots ==
1. By pressing the buttons you can remove the items from the menu

== Installation ==
1. Download the repository plugin
2. Activate the plugin from the WordPress menu
